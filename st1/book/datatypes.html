Standard datatypes

Standard datatypes

Using Booleans

The example below creates a Boolean instance with the value of true and assigns it to succeeded.

  succeeded = true;

Using Characters

The example below creates a Character instance with the value of 'a' and assigns it to a.

  a = 'a';

Using Natural numbers

The example below creates a Natural instance with the value of 21 and assigns it to age.

  age = 21;

Using Integers

The example below creates an Integer instance with the value of -1500 and assigns it to populationGrowth. 

  populationGrowth = -1500;

Using Rational numbers

The example below creates a Rational instance with the value of 1/2 and assigns it to half.

  half = 1 / 2;

Using Real numbers

The example below creates a real number with the value of the square root of 2 and assigns it to squareRootOf2.

  squareRootOf2 = new Real().sqrt(2);

The example below creates a real number with the value of Pi using a Real constructor and assigns it to pi.

  pi = new Real().pi();

Using Complex numbers

The example below creates a complex number with the value of 3 + 2i assigns it to complex.

  complex = new Complex().a(3).b(2);
