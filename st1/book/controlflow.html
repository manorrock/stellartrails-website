Controlling the flow

Controlling the flow

If then statement

The example below calls the isAdmin method on user to determine if an admin email should be sent. If it returns true then the admin email will be sent.

  if (user.isAdmin()) then {
    user.sendAdminEmail();
  }

If then else statement

The example below calls the isAdult method on person to determine if an acceptance email or a denied email needs to be sent.

  if (person.isAdult()) then {
    person.sendAcceptedEmail();
  } else {
    person.sendDeniedEmail();
  }

While statement

The example below calls the isNotEmpty method on stack to determine whether or not it is empty and if it is not empty it goes into the loop calling the pop method on stack.

  while (stack.isNotEmpty()) {
    stack.pop();
  }

Do while statement

The example below calls the decreaseByOne method on thermostat and then determines whether or not to terminate the loop by calling the isMoreThan10Degrees method on thermostat.

  do {
    thermostat.decreaseByOne();
  } while (thermostat.isMoreThan10Degrees());
